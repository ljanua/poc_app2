# 011 — SystemAdmin-controlled user assessor badge

## Idea

Add a **badge** attribute on each user, controllable **only by SystemAdmin**, to acknowledge users who have a **certificate** or **recognized experience** to assess player skills.

## Why it matters

Signals which assessors are credentialed, so coaches/clubs can trust skill ratings from qualified people.
